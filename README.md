# React + Vite + Tailwind CSS

Esta aplicação foi desenvolvida utilizando Vite, React e Tailwind CSS.

## Arquitetura e Técnicas utilizadas

Página desenvolvida em React onde utilizo os conceitos de componentização, props e renderização de elementos com ma(). O componente único 'App' reúne toda a lógica de definição e tratamento de estados entre os subcomponentes 'Tabuada' e 'NavButtons'. O resultado é uma página simples com um menu de botões laterais, cada opera uma função de mudança de estado ao clique, exibindo ao lado a tabuada completa do respectivo número.
